In according to the Sugar Labs
Licensing policy
, the list of licences that are allowed on
Activity Library
:
Software
Documentation
Content
Fonts
Be careful
Make sure that the
licence
key in the
activity.info
file contains as its value the same short name of the licence as shown in the aforementioned lists. Multiple licences are allowed and need to be separated by a semicolon symbol.
If your activity is covered by another one, please
let us know
.
