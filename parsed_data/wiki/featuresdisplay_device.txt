Summary
A frame device to control the display. The idea is to add their an option to change the brightness and to take a screenshot. Both actions are only available via the keyboard as of today.
Owner
Name: Martin Abente
Current status
Targeted release: 0.106
Last updated: 2015-05-15
Percentage of completion: 100%
Detailed Description
The display frame device will add an option to change the brightness of the display and to take a screenshot (alt+1). Both actions are only available via the keyboard as of today.
Benefit to Sugar
Expose features that are otherwise undiscoverable.  Portability to systems other than the XO laptop that have backlight brightness controls handled by GNOME or Unity desktops.
Scope
This Feature affects only the Sugar module. It is an extension.
