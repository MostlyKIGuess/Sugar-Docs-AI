Activity Library Users
If you are a user looking for cool new Activities please visit the
Sugar Activity Library
.
If you are looking for more information on how to use the Library please see
About the library
.
Activity Authors
If you are an Activity author and want to learn how to upload your Activity to
Activity author support
.
All activities uploaded to the Activity Library must be Free and Open Source Software. See our
Licensing
page for more information.
Library Editors
Library Editors are an important part of the Activities Team.  Editors review activities in the Library before they are available for general release.  Prior to review, newly uploaded activities are available in the
activities sandbox
.
If you are interested in learning more about becoming and Library Editor please see
Library Editors
.
Library Developers
If you are interested in learning how to become a Sugar Activity Library developer please see
ASLO Developers
.  Developers often refer to the Library as  ASLO.  Patches welcome!
Library Administrators
Library administrators keep the Library running smoothly.
If you are interested in learning how to become a Library Administrator please see
ASLO Administrators
.
Library Localizers
Library Localizers ensure that the Library and its Activities are translated for users around the world.
Please see
ASLO Localizers
.
