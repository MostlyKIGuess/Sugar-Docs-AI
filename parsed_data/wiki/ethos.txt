Description
Ethos is a small platforming game where players guide a non-player character through a treacherous world by drawing platforms with the mouse cursor. It is intended to eventually become a networked multiplayer title.
Current Status
Ethos is still in a very early stage of active development. Check back here, or its
github page
, later, to see how the project has progressed.
