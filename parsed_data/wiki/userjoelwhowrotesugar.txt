A graph showing total lines of code in Sugar since April 19 2006, when the first commit was made to Sugar's git repo.
Using
gitdm
, a set of scripts written to mine the Linux kernel's git
changelogs for contributor information, I have put together some numbers
on contributions to Sugar.
