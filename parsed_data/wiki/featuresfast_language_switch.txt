Detailed Description
Frame icon
The frame icon will contain two sections:
list of languages to fast switch to (with restarting Sugar Shell), all options need to have associated shortcuts to have the fastest switch;
list of languages to change current keyboard layout for, i.e., the visualization of Keyboard Control Panel section.
Set language on initial start
It will be useful to set the current language on initial Sugar start.
Configuration
The set of language will be based on Language and Keyboard Control Panel sections.
Benefit to Sugar
This feature is important for multilingual communities when it is useful to have the fast way to switch between languages, e.g., people are all time speaking in several languages and educational process is also basing on multilingual culture.
The one example is Peru, when people are speaking in Spanish and several local languages.
