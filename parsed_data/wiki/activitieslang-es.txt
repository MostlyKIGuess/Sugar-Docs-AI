Ejecutando Aplicaciones Linux Bajo Sugar
Existen dos formas de ejecutar aplicaciones Linux como Actividades bajo Sugar (lo que significa que no es necesario usar una terminal para cargarlas):
Sugar Coating
- Hace que las aplicaciones Linux se ejecuten bajo Sugar, pero sin modificarlas para tomar ventaja de las
capacidades colaborativas de Sugar.
Sugarización
- Aplicación Linux completamente integrada, rediseñada para tomar ventaja de las capacidades colaborativas de Sugar.
There are two ways to run Linux applications as Activities under Sugar (meaning you don't need to use Terminal to load them):
Sugar Coating
- Making Linux apps run under Sugar, but not modifying them to take advantage of Sugar's collaborative capabilities
Sugarizing
- A fully integrated Linux application, re-engineered to take advantage of Sugar's advanced collaboration abilities
Actividades Básicas de Sugar
Para ejecutar aplicaciones linux bajo Sugar (Por ejemplo, para hacerlas correr en la interfaz Sugar, sin la colaboración y otra integración de Sugar dentro de ella), lea estas instrucciones en
Ejecutando Aplicaciones Linux Bajo Sugar
.
To run Linux apps under Sugar (i.e. making them to run in the Sugar interface, without the collaboration and other Sugar integration programmed into them), read these instructions on
Running Linux Applications Under Sugar
.

VLC Media Player
VLC Media Player es un reproductor multimedia altamente portable para varios formatos de audio y video (MPEG-1, MPEG-2, MPEG-4, DivX, mp3, flv, avi, ogg, ...), asi como DVDs, CDs y varios protocolos de streaming. Puede usarse también como un servidor de streaming en unicast o multicast para IPv4 o IPv6 en una red de banda ancha. El reproductor VLC es considerado mucho mejor que el Mplayer por algunos usuarios porque ofrece más escenarios, opciones y características. No es necesario ejecutarlo en una terminal. Lea
Actividad VLC
para aprender como instalarlo.

The VLC media player is a highly portable multimedia player for various audio and video formats (MPEG-1, MPEG-2, MPEG-4, DivX, mp3, flv, avi, ogg, ...) as well as DVDs, VCDs, and various streaming protocols.  It can also be used as a server to stream in unicast or multicast in IPv4 or IPv6 on a high-bandwidth network.; the VLC Player is considered to be better by some users then MPlayer because it offer more settings, options and features. There is also no need to run it from Terminal. Read
Activities/VLC
to learn how to install it.
Seamonkey
SeaMonkey ha heredado el éxito de "todo en uno" el concepto original de Netscape Communicator y que sigue la línea de productos basados en lo moderno, arquitectura multiplataforma proporcionada por el proyecto Mozilla.
El navegador de Internet como el corazón de SeaMonkey utiliza el mismo motor de renderizado como su hermano Mozilla Firefox, con características populares como la navegación por pestañas, bloqueo de popup, completar a medida que escribe y muchas otras funciones para una buena experiencia en la web.
Correo SeaMonkey y el cliente lector de noticias comparte mucho codigo con Mozilla Thunderbird y caracteristicas adaptables para filtrado de correo no deseado, las etiquetas de correo y opiniones, manejo de varias cuentas, S / MIME, libretas de direcciones con apoyo de LDAP y está listo para uso del sector privado y empresarial.
Componentes adicionales que incluyen un fácil uso del editor HTML, La aplicación ChatZilla IRC y herramientas para el desarrollo web como Inspector DOM y el Debugger de JavaScript.
Si esto es aún insuficiente,SeaMonkey puede ser extendidos con numerosos add-Ons que proporcionan funcionalidad adicional y personalizacion para una completa experiencia en internet
SeaMonkey puede ser considerado en Sugar
SeaMonkey has inherited the successful all-in-one concept of the original Netscape Communicator and continues that product line based on the modern, cross-platform architecture provided by the Mozilla project.
The Internet browser at the core of the SeaMonkey suite uses the same rendering engine as its sibling Mozilla Firefox, with popular features like tabbed browsing, popup blocking, find as you type and a lot of other functionality for a smooth web experience.
SeaMonkey's Mail and Newsgroups client shares lots of code with Mozilla Thunderbird and features adaptive Junk mail filtering, labels and mail views, multiple accounts, S/MIME, address books with LDAP support and is ready for both private and corporate use.
Additonal components include an easy-to-use HTML Editor, the ChatZilla IRC chat application and web development tools like a DOM Inspector and a JavaScript debugger.
If that's still not enough, SeaMonkey can be extended with numerous Add-Ons that provide additional functionality and customization for a complete Internet experience.
Seamonkey can be Sugar-coated.

Totem Media Player
Totem Mediaplayer esta actualmente instalado en la OLPC XO-1. Es motor que soporta la multimedia en el sistema. Sin embargo, no soporta MP3 o video MPEG-4. Un dirver gratuito puede descargarse para adicionar compatibilidad con MP3. Puede aplicarse la inclusión en sugar para acceder al programa desde el panel de actividades.
The Totem media player is already installed in the OLPC XO-1. It is the engine that supports media on the system. Out of the box it does not support MP3 music or MPEG-4 video. A free driver can be downloaded to add MP3 compatability. A Sugar-coating can be applied to easily access the program from the Activities panel.
