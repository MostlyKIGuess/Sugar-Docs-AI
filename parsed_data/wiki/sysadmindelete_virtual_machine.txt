Title: Sysadmin/Delete virtual machine
URL: https://wiki.sugarlabs.org/go/Sysadmin/Delete_virtual_machine
--------------------------------------------------------------------------------
Delete virtual machine on
Machine/freedom
or
Machine/justice
virsh destroy VMNAME
virsh vol-delete --pool justice /dev/justice/VMNAME
rm /etc/libvirt/qemu/VMNAME
delete DNS record (
Service/Nameservers
)
delete backups from
Service/backup

Sysadmin/Add virtual machine
