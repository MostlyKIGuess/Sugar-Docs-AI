Instrucciones para actualizar el firmware
).
ADVERTENCIA:
esta acción sobrescribirá todos los datos en la NAND de su OLPC XO-1!
En un sistema Fedora 10:
sudo yum -y install crcimg mtd-utils
En Debian o Ubuntu:
sudo apt-get install mtd-utils
crcimg no es empaquetado para Debian (que yo sepa), así que se tendrá que compilar:
wget
http://dev.laptop.org/~mstone/releases/SOURCES/crcimg-1.1.tar.bz2
tar xvf crcimg-1.1.tar.bz2
cd Makefile.build
sudo make -f Makefile.build install
Siguiente:
wget
http://dev.laptop.org/~cjb/rawhide-xo/livecd-iso-to-xo.sh
wget
http://download.sugarlabs.org/soas/snapshots/2/Soas2-200906221314.iso
Ejecute este comando (o similar, dependiendo del número de versión de su iso):
sudo sh livecd-iso-to-xo.sh Soas2-200906221314.iso Soas.img
y, a continuación, copia Soas.img y Soas.crc a una llave USB o tarjeta SD.
En la OLPC XO-1, arranque con la llave USB conectada y las cuatro claves del juego de prensa para obtener la pronta Aceptar en Open Firmware (OFW).
Escriba los siguientes comandos:
disable-security
copy-nand u:\Soas.img
Nota: dependiendo del sistema de archivos en tu llave USB, tal vez los nombres esten truncados
dir u:\
copy-nand u:\SOAS-2~1.img
Debería ver una animación de los bloques que se escriben en el NAND. Cuando esté completa, usted estará de regreso en el símbolo ok.
Escriba:
reboot
y el OLPC XO-1-se reiniciará y se debe estar ejecutando Azúcar 0.84.
