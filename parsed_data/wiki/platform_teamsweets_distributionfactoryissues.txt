Long start on XO-1
If Sugar Network integration is enabled, the starting up process of Sugar Shell will be much longer than usual. This time should be decreased in further versions.
Live-time update
Current Sugar Network client applications doesn't reflect on server side changes. So, if there are changes made by another users, or, in some cases local changes, F8 screen needs to be refreshed. Open context menu and select
"Refresh"
.
Cannot connect to secured Access Points
Factory repository contain Sugar-0.94 which is highly depends on Network Manager DBus API. In recent systems, there is no way to connect to wireless Access Points required authentication.
