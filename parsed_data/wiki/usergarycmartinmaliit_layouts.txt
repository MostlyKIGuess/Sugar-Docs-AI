Maliit Layouts
This is the set of Maliit language layouts provided by default and modified for a clean grey visual style to tie in better with the Sugar look and feel, layouts have been corrected/cleaned up (over/under flow of screen, spacing and alignment issues), and a close keyboard key added to all layouts. The image history acts as a 'before' snapshot while cleaning up layouts to make sure we don't regress. Symbol pages (123?) are shown though not for every language as many are duplicates of previous symbol layouts. See the
target designs
for the current layout and style goals. For past physical keyboards see
