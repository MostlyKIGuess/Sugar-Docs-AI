The current version is Skolelinux 6.0.4 Edition.
See the most recent release notes, dated March 11th, 2012:
http://lists.debian.org/debian-edu-announce/2012/03/msg00001.html
The download and install instructions are comprehensive.
http://www.slx.no/en/downloads
Versions of Sugar offered
There is no sugar installed by default. Sugar versions 0.84, 0.86, and 0.88 are available through Synaptic Package Manager. A brief test (3 July 2012) produced Sugar 0.88.1 working fine in an emulator.
