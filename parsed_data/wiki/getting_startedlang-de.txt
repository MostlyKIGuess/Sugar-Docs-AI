Über Sugar
Template:Transaleted text
Sugar ist eine Desktopoberfläche die auf Microsoft Windows, Apple's OS X sowie GNU/Linux aufsetzt.
Es ist konzipiert als ein Tool, das Kindern das interaktive Lernen erlaubt. Eines der ersten Dinge die ein Kind sieht, ist nicht eine Festplatte oder Müll wie andere Kinder in der Nachbarschaft. Programme und Applicationen werden "Activities" genannt, viele von ihnen erlauben
zusammenarbeit
mit anderen Kindern in der Umgebung via
Wi-Fi
oder über ein
Jabber Netzwerk
. Sugar setzt in seinen "Activities" auf zusammenarbeit die automatisch aktiviert werden.
Sugar is developed in
Python
and runs on a
GNU/Linux
Kernel
, originally from the
Fedora Project
, and now from a
variety
of GNU/Linux distributions.
For an overview of the components composing a Sugar system see the
Sugar System Stack
.
Sugar ist programmiert mit
Python
und läuft mit dem
GNU/Linux
Kernel
, im original vom
Fedora Projekt
, eine Distribution von GNU/Linux.
Für eine Übersicht von den Komponenten die zum Sugar System gehören gehen Sie bitte auf
Sugar System Stack
Sugar bekommen
Gehen Sie hierfür bitte zu
Downloads
.
For more technical information, please see the
Supported systems
page for a list of the different ways that Sugar may be run on a computer.
Für mehr technische Informationen, finden sie
hier
die Vorraussetztungen die Ihr Computer erfüllen muss um Sugar auf im zu starten.

Internetverbindung
Connecting to the Internet is something that somewhat falls between the cracks of Sugar and the computer it's running on.
On the XO laptop, there are three ways to connect to the Internet:
Wireless access point (Wi-Fi hotspot);
“School Server” mesh network; or
“simple” mesh network, which lets you collaborate directly with other XOs running Sugar.
From a conventional laptop running Sugar, connecting through an access point works. (Mesh support is becoming available on more machines.) Depending upon which
Jabber server
you are connected to, you'll see different collections of "neighbors" in the Neighborhood View.
Read the
Connecting to the Internet page
for detailed instructions.
So stellen Sie eine Internetverbindungen mit Sugar her.
Auf dem XO Laptop gibt es drei Varianten sich mit dem Internet zu verbinden:
WLAN / WiFi hotspot;
+"Schulserver" mesh Netzwerk; oder
"einfaches" mesh Netzwerk, welches dir ermöglicht direkt mit anderen XO's über Sugar zusammen zu arbeiten.
Von einem konventionellen Laptop Kann Sugar über ein normales Netzwerk bzw. WLAN mit dem mesh Netzwerk über den Router kommunizieren. Über den
Jabber Server
sind Sie mit vielen anderen Nachbarn via Internet verbunden.
Lesen Sie bitte hierzu
Connecting to the Internet page
(englisch) für detailierte informationen.
